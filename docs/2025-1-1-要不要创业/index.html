    2025-1-1 要不要创业

      • 一、创业者最核心的一种特质：韧性
      • 二、关于启动创业的两条建议
      • 总结
    • （一）韧性才是创业者最重要的特质
    • （二）如何判断是否有足够的韧性
    • 二、关于启动创业的两条建议
    • （一）不要将找创业想法和联合创始人当作独立的两件事
    • （二）通过将想法转化成副业项目，创造开启创业的时机

    一、创业者最核心的一种特质：韧性 #

    （一）韧性是创业者成功的关键 #

    1. 创业者类型多样化 成功的创业者不仅限于才华横溢的程序员或充满魅力的产品天才。不同类型的人都能通过各自的优势取得成功。
    2. 韧性的重要性 创业初期面临的拒绝和挫折是对个人心理承受能力的巨大考验。韧性帮助创业者坚持下去，获得早期用户和市场认可。

    （二）如何判断自己是否具备韧性 #

    1. 自信与韧性并不等同 自信的创业者未必能应对创业中的长期挑战，而看似不自信的人可能拥有强大的韧性。

      案例：Benchling 创始人 Sajith

      • 虽初期表现内向、不擅长销售，但他克服重重困难，带领公司成为行业领先者。

    2. 创业动机的灵活性

      • 初期动机（如追求财富或好奇心）并不决定成功。
      • 真正重要的是对解决问题的兴趣和与团队合作的愉悦感。

    3. 应对最坏情况

      • 在创业前，评估失败后可能的代价与学习价值，确保自己心理和经济上能承受。
      • 失败的创业经历同样能提升职业竞争力，为未来打下基础。

    二、关于启动创业的两条建议 #

    （一）将寻找创业想法和联合创始人结合起来 #

    1. 通过讨论激发创意 与志趣相投的人共同探讨问题和解决方案，能更高效地产生可行的创业点子。
    2. 选择环境和人脉
      • 如果当前圈子无法提供支持，考虑加入初创公司工作，积累经验和潜在合作伙伴。

    （二）通过副业项目开启创业之路 #

    1. 从副业项目开始尝试
      • 将点子转化为简单的初版产品，获取早期用户反馈并逐步优化。
    2. 衡量项目潜力
      • 优先关注少数用户的强烈热爱，而非普遍接受度。
    3. 注重自身感受
      • 判断是否享受创造过程，与日常工作对比看是否更有激情。
    4. 评估合作关系
      • 如果与合作者合作愉快且愿意共同创业，这是极为宝贵的创业起点。

    总结 #

    创业是一条需要韧性、勇气和策略的旅程。通过正确评估自身特质、动机和环境，并通过实践验证想法，创业者能更好地适应挑战，最大化创业成功的可能性。

    一、创业者最核心的一种特质 #

    （一）韧性才是创业者最重要的特质 #

    如果你对成功创业者的印象仅基于书籍或电影中对知名创始人的描绘，比如他应该是像《社交网络》中马克·扎克伯格那样，是一名无情而聪明的程序员，或者像许多电影中描绘的史蒂夫·乔布斯那样，是一个充满魅力的产品天才，那么你很可能会误认为，如果你不是这样的人，就很难成为一名成功的创业者。 现在，我不否认，一名才华横溢的程序员或有魅力的产品天才肯定会更容易成为一名成功的创始人。但是我可以说，在与许多成功的创始人合作之后，我发现在创业公司取得成功的人有很多种类型，他们拥有不同的优势，而不仅仅是那些刻板印象中的特点。

    作为一名投资者，我的工作是选择哪些人会成为出色的创始人。然而，即使在经过了15年的创业和投资之后，我对于谁会成为出色的创始人仍然感到惊讶。这意味着作为创始人的你更难确定：自己是否适合做一名创业者。

    你可能会认为，只要简单地看看某个人在学校或工作中表现得有多好，就可以初步预测将来他的创业公司的成功概率。这确实是我在2010年刚开始在 YC 工作并阅读 YC 创业营申请表时的想法。然而，在与更多的创始人合作之后，我意识到这些信号（某个人在学校或工作中表现）并没有我预期的那么重要。

    当你创业时，有时仅仅是为了说服一个用户去尝试你的产品，你都必须付出大量的心血、汗水甚至眼泪，更别说去说服前10个或100个用户了。你必须经历许多拒绝才能达到这个目标。而且因为这是你的创业公司，所有的拒绝都以一种非常个人化的方式影响着你。然而当你在某家公司工作时，被拒绝却并不会有这种感觉。创始人需要很强的韧性才能在初期的困难中坚持下来，获得早期用户。这是我认为创业公司创始人最重要的品质就是韧性（resilience）。具备韧性的人，适合创业，并值得去尝试创建一家公司。

    （二）如何判断是否有足够的韧性 #

    如何知道自己是否具备足够的韧性去创业呢？ 1.自信的创业者不一定有韧性 当我刚开始在 YC 工作时，我认为自信可以成为衡量一个人是否有足够韧性的标准。比如在 YC 面试过程中，如果一个创始人表现得很自信，说话非常有说服力、充满能量，那么我们就会认为他很可能是有韧性的创业者。然而，随着我合作的创业者越来越多，我意识到这是错误的观点。 后来的事实证明，一个人最初可能显得非常自信，但当他们的创业公司遇到困难时却可能显得没有韧性。相反，一些看起来很安静、不自信的创业者反而是比较有韧性的创业者，他们的公司最后也成功上市。

    在这里，我将向你分享 Benchling 的创始人 Sajith 的创业故事。 我们在 2012 年投资了 Benchling，现在它的市值已经超过60亿美元，Sajith 也成长为一个很出色的创始人和 CEO。但是，我记得我们第一次面试他和他的联合创始人 Cory 时，他们都是声音很小很轻的工程师。我们当时都担心他们不适合在 Benchling 工作，因为 Benchling 公司的创业想法是向生物技术公司和制药公司出售软件，我们不确定他们是否擅长这种企业销售工作。在 Demo Day 上，新的投资者们也有同样的担忧，担心他们无法让这些生物技术公司或制药公司为 Benchling 的软件付费。因此，Benchling 在 Demo Day 上筹集资金时非常困难。 后来的事实也证明，销售对 Benchling 确实是一个真正的挑战。从他们在 YC 毕业之后的一年多里，尽管他们能够拥有一些充满热情的免费用户，并且他们的产品不断在改进，但 Benchling 仍然没有实现任何收入。直到再经过了将近一年的时间，Benchling 才开始产生收入。如今，Benchling 已经发展为一家大的生物科技软件公司，并拥有许多世界顶尖的生物技术和制药公司作为客户。 回顾过去，我始终相信 Sajith 是一位优秀的工程师，但当我们在2012年投资他时，我无法预测他是否有足够的韧性克服如此多的的拒绝和挫折，并成为一个需要多年才能精通企业销售的公司创始人。 这里我并没有分享其他创始人的故事，是因为这些创始人在我们最初投资他们时同样显示出才华，但最终却没有像 Sajith 那样具备足够的韧性走得更远。

    2.创业动机强不代表韧性强 我曾经认为拥有合适的创业动机的人更有可能拥有韧性。然而，随着时间的推移，我逐渐发现最初的动机并不像我曾经认为的那么重要。 我认为创业有很多完全合理的动机。例如为了追求财富而创业，虽然流行的说法是“不应该仅仅为了赚钱而创业”，但我认为，这是可以接受的。创业是改变人生的一种方式，而如果渴望财富让你开启行动，那就大胆追求吧。我还认为，出于好奇心而创业也是一个不错的创业动机。 然而，我有信心认为，最初的动机并不那么重要，因为我亲眼见证了动机随时间而改变的情况。有些创始人最初动机是创立公司然后一年内卖出去，但随着时间的推移，他们改变了想法。很久以后，他们仍然投入在这家创业公司，而这家公司已上市。 要在创业公司中持续努力并度过艰难时期，你需要坚持不懈的动力。我认为最好的动力是对你正在解决的问题真正感兴趣，并喜欢与你一起工作的人（我们稍后会更多地讨论如何找到这些人）。 从分析创业失败最坏的结果来判断自己是否有韧性去创业 如果你对创业感到好奇并准备尝试，那么你首先要问自己一个非常实际的问题：“我现在有什么可以失去的吗？”这不是一个修辞性问题，而是一个实际的思考。 设想一下创业失败的最坏情况，看看自己是否能够承受。创业可能需要至少一年的时间来积累足够的数据，以评估你所从事的项目是否有前景。在最坏的情况下，你可能需要在那一年内关闭公司，并且可能没有或只有很少的薪水。你能够承受这种最坏情况吗？ 这个问题的答案因人而异。如果你即将大学毕业，花一年时间创业并不是什么大不了的事情，因为一年后仍然会有其他工作机会等待着你。但如果你在过去的9年里一直是一家顶级科技公司的员工，并且今年有可能晋升到重要职位，那么花一年时间创业可能会让你失去很多。 问问自己是否能应对最坏情况，这是了解自己是否应该创业的最好方式。要对自己诚实，如果你实际上无法承受最坏情况，那么你自己的焦虑感可能会破坏你对创业的投入。

    失败的创业经历也具有价值：

    失败的创业经历具有学习价值。当你考虑创业的最坏情况时，还有一件事情也值得考虑，那就是创业能够带给你很多学习的机会。在进行最坏情况分析时，很少有人能够意识到这种学习的价值。

    作为创始人，你需要负责推动公司所有的事务，这意味着你需要同时处理多种不同类型的工作，例如兼顾销售、产品和客户支持等工作。最终，即使创业不成功，这个过程帮助你弄清楚了自己真正喜欢什么类型的工作。 例如，有些人在经历了一次不成功的创业后直接创办了另一家公司，因为他们从之前的失败中学到了宝贵的经验教训。而另一些人则意识到自己其实非常喜欢专注于某个特定领域的工作，于是他们选择加入其他公司。 创业经验可以大大增加职业机会。我的第二个创业公司 Triplebyte 利用软件帮助公司找到并雇用优秀人才，我与数百家雇主合作，帮助他们寻找最优秀的人才。在与数百家雇主合作的过程中，我发现许多雇主都明确表示他们希望找到有创业经验的人，因为创过业的人能够发起和领导项目。 一些最成功的 YC 公司实际上已经根据这一观点来建构他们的招聘流程。例如 Rippling，这是由 Parker Conrad 创立的一家百亿美元公司，也是 YC 投资的公司。他们聘请了前创业公司的创始人，并将他们安排在产品部门的领导岗位上，让他们像总经理一样运营特定的产品线。Rippling 在招聘方面取得了巨大成功，目前有大约50名创过业的人在该公司工作。 在 YC，我们经常看到创业公司失败后，创始人加入另一个 YC 公司的案例。例如 Nick Grandy，他的第一家创业公司 Wonder Bar 在2008年获得了 YC 的投资。后来公司倒闭了，但他加入了另一家 YC 公司 Airbnb 并成为了它的一号员工，这当然非常成功。在 Airbnb 工作多年后，他又离开并成立了另一家 YC 公司，现在价值超过30亿美元。

    二、关于启动创业的两条建议 #

    （一）不要将找创业想法和联合创始人当作独立的两件事 #

    首先，我要鼓励你不要把寻找创业想法和联合创始人看作是两个独立的任务。这是因为自己独立想出好想法是很困难的。创业想法一开始只有一个模糊的方向，更像是一种直觉或冲动，而不是一个清晰的构想。你需要通过多次完善来将其转化为更具体的想法。因此，有人能够与你讨论最初的想法，进行辩论是非常重要的。而最佳人选就是你的联合创始人。我发现，我最好的创业想法都是在与聪明人交谈时想到的。 如何同时寻找创业想法和联合创始人：

    1.考虑你认识的人中，那些你喜欢与之讨论想法的人 我建议你考虑一下你认识的人中，那些你喜欢与之讨论想法的人。在大学里，这些人通常是你在遇到难题时会去找的人，或者他们是你最喜欢一起做小组作业的人。在工作中，这些人是你觉得最能提高你工作效率和帮助你最好地完成工作的同事。我建议你与这些人交流，谈谈你觉得有趣的想法或技术，讨论一下你每天使用的产品、你欣赏或不满意的产品，还可以讨论一下你认为今天被低估的技术及其原因。

    2.阅读和研究自己感兴趣的主题 在你与他人讨论想法的同时，抽出一些时间来阅读和研究你感兴趣的主题。如果你是一个好奇心旺盛的人，这应该是一个自然而然的行为。所有这些都会让你的思维更容易注意到和想到潜在的创业想法，并且它还可以帮助你更好地了解你的朋友或同事，看他们是否适合成为你的联合创始人。

    3.到初创公司工作 如果你正在为找人讨论点子而苦恼，那么你可能需要改变环境，结识一些新朋友。如果你真的有意创业，最好选择一个你可能会遇到许多潜在合作伙伴的环境。因此，最适合的方式是在初创企业工作。在那里，你可以从内部了解创业公司的运作，并且你的同事们通常比普通公司的员工更倾向于冒险。所以，如果你还在上大学，我建议你毕业后找一家创业公司工作。如果你已经在一家大公司工作，可以考虑离开并加入一家创业公司，以此作为创业的跳板。

    （二）通过将想法转化成副业项目，创造开启创业的时机 #

    一旦你周围有聪明的人，并且你愿意和他们讨论想法，我建议你开始尝试把这些想法转化成副业项目。在这些对话中的某个时刻，你可能会说出类似于“如果有人创造出……产品就很酷”的话，这时你就应该停下来思考如何成为创造这个产品的人。然后，你可以尝试在周末制作出简单的第一版，或者找到潜在用户，向他们介绍这个想法，看看他们的反应。 不断地重复这个步骤，直到你制定出将这个想法转化为现实的计划，无论这个想法目前多么小。因为你并不是在试图推出下一个独角兽企业，你只是要逐渐适应将想法转化为现实的过程，体验到推出某个东西的感觉。

    1.具备实际构建它们所需的技能 要将你的想法变成现实，你需要具备实际构建它们所需的技能，比如，学习编程或者找到一位程序员共同创业。如果你不是程序员，那么你需要学习编程或者找到一位程序员共同创业。从一个没有编程经验而开始创业的人的经验来看，我建议你你开始学习编程，但只需学到足够支持你构建基础产品即可。

    2.理性分析副业项目进行创业转化的前景 当你与聪明的人一起工作并开始从事一些副业项目时，这是否是你离职创业的理想时机呢？ 如果你的副业项目迅速发展，这足以为你做出创业决定提供很好的支持。但如果你认真考虑将来创办一家公司，就不能仅仅依赖这种情况。因为即使你启动了副业项目并获得了一些早期用户的反馈，这些反馈也会是复杂多样的，原因是早期用户对产品的反应往往不尽相同。 为了评估副业项目作为创业公司前景的可行性，我想分享一些来自 YC 创始人 Paul Graham 的建议。他说，“创造一个少数人无比热爱的产品，好过于创造一个多数人都一般热爱的产品”。如果你的副业项目得到少数人强烈的认可，比如他们真心地认为有了这个产品的存在，他们能做一些非常不同的事情，那么你正在开发的产品可能的确很有潜力。

    3.关注自己对副业项目的享受程度 我给出的主要建议是不要过分关注副业项目的进展，而是是否享受将想法转化为实际产品的过程。你是否正在学习新事物并对此充满激情？将这种感觉与你的日常工作进行对比。如果你发现工作让你感到疲惫，但你在晚上和周末从事副业项目时有更充沛的精力，那么这可能是时候离职创业了。

    4.关注自己与合作者合作的乐趣程度 要特别关注你与副业项目合作者一起工作的快乐程度。如果你喜欢与他们一起工作，并且你们都想成为创业公司的创始人，你要认识到这是一个十分难得的机会。因为，找到一个可以一起创建公司的优秀联合创始人，就是离职创业的绝佳理由了。
